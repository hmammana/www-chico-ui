ch.Carousel (Class)

Documentation updated on Tue Jul 30 2013, generated by JsDoc Toolkit.

Overview

ch.Carousel extends ch.Widget. Carousel is a large list of elements. Some elements will be shown in a preset area, and others will be hidden waiting for the user interaction to show it.

Returns

itself

Use

Create a Carousel without configuration.

var foo = $("#example").carousel();

Create a Carousel with configuration parameters.

var foo = $("#example").carousel({
    "width": 500,
    "pagination": true,
    "arrows": "over"
});

Create a Carousel with items asynchronously loaded.

var foo = $("#example").carousel({
    "asyncData": [
        {"src": "a.png", "alt": "A"},
        {"src": "b.png", "alt": "B"},
        {"src": "c.png", "alt": "C"}
    ],
    "asyncRender": function (data) {
        return "";
    }
});

Configuration

The following options lets you create a customized ch.Carousel:

  • conf: Object Object with configuration properties.
  • conf.width: Number || String Sets width property of the component's layout. By default, the width is elastic.
  • conf.pagination: Boolean Shows a pagination. By default, the value is false.
  • conf.arrows: Boolean Shows arrows icons over or outside the mask. By default, the value is "outside".
  • conf.asyncData: Array Defines the content of each item that will be load asnchronously as array.
  • conf.asyncRender: Function The function that receives asyncData content and must return a string with result of manipulate that content.
  • conf.fx: Boolean Enable or disable UI effects. By default, the effects are enabled.
  • conf.maxItems: Number (Since 0.10.6) Set the max amount of items to show in each page.
  • conf.page: Number (Since 0.10.6) Initialize the Carousel in a specified page.

Methods

arrows(config)

Allow to manage or disable the "Next" and "Previous" buttons flow ("over" the mask, "outside" it or "none"). (Since 0.10.6).

Parameters

  • config Since 0.10.6: String || Boolean CSS transition properties. By default it's "outside".

Returns

  • Chico UI Object

Examples

Put arrows outside the mask.

foo.arrows("outside");
// or
foo.arrows(true);

Put arrows over the mask.

foo.arrows("over");

Disable arrows.

foo.arrows("none");
or
foo.arrows(false);
