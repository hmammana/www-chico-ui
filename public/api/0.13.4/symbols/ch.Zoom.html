ch.Zoom (Class)

Documentation updated on Tue Jul 30 2013, generated by JsDoc Toolkit.

Overview

ch.Zoom extends ch.Floats. Zoom shows a contextual reference to an augmented version of main declared image.

Requires

  • ch.onImagesLoads

Returns

itself

Use

Create a Zoom component wrapping the original image with a anchor element pointing to a bigger version than the original.

var widget = $(".example").zoom();

Configuration

The following options lets you create a customized ch.Zoom:

  • conf: Object Object with configuration properties.
  • conf.fx: Boolean Enable or disable fade effect on show. By default, the effect are disabled.
  • conf.context: Boolean Sets a reference to position of component that will be considered to carry out the position. By default is the anchor of HTML snippet.
  • conf.points: String Sets the points where component will be positioned, specified by configuration or "lt rt" by default.
  • conf.offset: String Sets the offset in pixels that component will be displaced from original position determined by points. It's specified by configuration or "20 0" by default.
  • conf.content: String This message will be shown when component needs to communicate that is in process of load. It's "Loading zoom..." by default.
  • conf.width: Number Width of floated area of zoomed image. Example: 500, "500px", "50%". Default: 350.
  • conf.height: Number Height of floated area of zoomed image. Example: 500, "500px", "50%". Default: 350.
