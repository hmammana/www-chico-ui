ch.Tabs (Class)

Documentation updated on Tue Jul 30 2013, generated by JsDoc Toolkit.

Overview

ch.Tabs extends ch.Widget. Tabs lets you create tabs for static and dynamic content.

Returns

itself

Use

Create a new Tab Navigator without configuration.

var widget = $(".example").tabs();

Create a new Tab Navigator with configuration.

var widget = $(".example").tabs({
    "selected": 2
});

Configuration

The following options lets you create a customized ch.Tabs:

  • conf: Object Object with configuration properties.
  • conf.selected: Number Selects a child that will be open when component was loaded. By default, the value is 1.
